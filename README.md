# Railway Reservation System

This is a simple Java console-based project designed for beginners.
The project simulates a basic railway ticket booking system.

## Objective
To understand basic Java concepts like variables, loops, conditional statements,
and user input using Scanner.

## Features
- Book Ticket
- View Ticket Details
- Cancel Ticket
- Check Available Seats
- Exit System

## Technologies Used
- Java
- GitHub

## How the System Works
1. User selects an option from the menu
2. System performs booking, viewing, or cancellation
3. Seat count is updated automatically
4. Output is displayed on the console

## How to Run the Project
1. Open Main.java in any Java IDE
2. Run the program
3. Enter menu choices as shown

## Conclusion
This project helps beginners understand how real-life systems
can be implemented using simple Java logic.
